# Sistema de Atendimento Inteligente - SENAI Gama

Sistema de gerenciamento de filas desenvolvido para otimizar o atendimento presencial.

## 🚀 Tecnologias
- **Backend:** FastAPI (Python)
- **Banco de Dados:** MySQL
- **ORM:** SQLAlchemy
- **Frontend:** HTML/CSS/JS (Vanilla)

## 📊 Diferenciais
- **Cálculo de Tempo de Espera:** Monitora o tempo real de cada usuário na fila.
- **Histórico de Atendimento:** Persistência de dados para análise de performance.
- **Rota de Estatísticas:** API pronta para dashboards de BI.

## ⚙️ Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Configure o MySQL e o arquivo `database.py`.
3. Inicie o servidor: `python main.py`
